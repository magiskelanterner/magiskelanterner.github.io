Quick Start

Build your first project in 3 steps.

1. Initialize

npx projectname init my-app
cd my-app

2. Start the dev server

npm run dev

3. Open your browser

Visit http://localhost:3000 and you should see the welcome page.

Tip Run npx projectname --help to see all available CLI commands.

Configuration

Customize behavior through the config file.

Create a project.config.js in your root directory:

// project.config.js
export default {
  port: 3000,
  output: './dist',
  plugins: [],
  theme: {
    primary: '#4361ee',
  },
};

Options

OptionTypeDefaultDescription
portnumber3000Dev server port
outputstring"./dist"Build output directory
pluginsarray[]List of plugins to load
themeobject{}Theme customization
Warning Changing the output path after your first build may leave stale files in the old directory.

API Reference

Core functions and their signatures.

init(options)

Initialize a new project instance.

import { init } from 'projectname';

const app = init({
  port: 3000,
  debug: true,
});
ParameterTypeDescription
options.portnumberServer port (default: 3000)
options.debugbooleanEnable debug logging

app.use(plugin)

Register a plugin with the application.

import { myPlugin } from 'projectname-plugin';

app.use(myPlugin({ option: true }));

app.start()

Start the application server. Returns a Promise.

await app.start();
console.log('Server running');
